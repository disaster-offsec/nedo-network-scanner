# 🔍 Port Scanner

Многопоточный сканер TCP-портов на C++ с поддержкой баннер-граббинга, OS fingerprinting и экспортом результатов.

## ✨ Возможности

- **Сканирование портов** — одиночные, диапазоны, все 65535 портов
- **Многопоточность** — динамическое распределение нагрузки (до 200 потоков)
- **Баннер-граббинг** — SSH, HTTP (с отправкой HEAD-запроса)
- **OS Fingerprinting** — определение ОС по TTL, Window Size и TCP-опциям
- **Экспорт результатов** — JSON, CSV, TEXT
- **Гибкий CLI** — управление таймаутом, количеством потоков и форматом вывода
